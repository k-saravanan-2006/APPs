
# Shoe Shop App

A modern and user-friendly Shoe Shopping Application built using Flutter. This app allows users to browse shoes, view product details, and add items to cart with a smooth and responsive UI.


## Features

✅ Beautiful & clean shopping UI

✅ Product listing screen

✅ Product details page

✅ Add to cart functionality

✅ Responsive layout for all devices

✅ Built with Flutter Material Design


## Tech Stack

Flutter

Dart

Material UI


## Installation

Clone the repository

git clone https://github.com/k-saravanan-2006/shop_app.git


Open the project folder

cd shop_app


Install dependencies

flutter pub get


Run the app

flutter run

## App Flow

User opens app and sees shoe products

Select a shoe to view details

Click Add to Cart

Cart page shows selected items

User can continue shopping
